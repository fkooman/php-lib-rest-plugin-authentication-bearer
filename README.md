# Introduction
This is a plugin for supporting HTTP Bearer Authentication in your REST service 
based on `fkooman/rest`.

# Installation
You need to require `fkooman/rest-plugin-bearer` in your `composer.json` and 
run `composer update`.

# Usage
See `example` directory.
